Pixel Button

The PixelButton is the button component from the Vue-Pixel library.

Usage

<!-- Usage example -->
<pixel-button raised accent>Accent</pixel-button>

Accent

API

Props

Name Description Default Example
flat Renders a flat button Flat Button
outlined Renders a outlined button Outlined Button
raised Renders a raised button Raised Button
disabled Makes the button unclickable Disabled Button
rounded Makes the button rounded Rounded Button
stripped Makes the button stripped Stripped Button
autofocus Sets focus on page load Autofocus Button
uppercase Sets the button text as uppercase Uppercase Button

Events

Name Description Detail
click When user clicks on button ClickEvent
focus When button receives focus FocusEvent
blur When button looses focus FocusEvent
mouseover When user mouse overs the button MouseEvent
mouseout When user mouse outs the button MouseEvent

Theming

Name Description Example
color Sets the button color Custom Color
background-color Sets the button background color Custom Background Color
border-color Sets the button border color Custom Border Color
